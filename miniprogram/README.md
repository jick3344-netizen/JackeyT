# 老黄的品牌三问小程序

这是原生微信小程序工程，用户端包含微信登录、个人/企业品牌问卷、报告生成提示、工作台与完整报告。

## 在微信开发者工具中打开

1. 打开微信开发者工具，选择“导入项目”。
2. 项目目录选择本文件夹：`miniprogram`。
3. 填入你在微信公众平台创建的小程序 AppID；开发阶段可选择测试号。
4. 在 Netlify 的环境变量中添加：

   - `WECHAT_MINIPROGRAM_APP_ID`：微信小程序 AppID
   - `WECHAT_MINIPROGRAM_APP_SECRET`：微信小程序 AppSecret

5. 在微信公众平台的“开发管理 - 开发设置 - 服务器域名”中，将下列地址加入 `request` 合法域名：

   `https://laohuang-brand-three-questions.netlify.app`

6. 编译后，点击首页的“微信登录并进入”开始测试。

## 发布前检查

- 在微信公众平台完善小程序名称、头像、服务类目、隐私保护指引与用户协议。
- 用真机测试微信登录、问卷自动保存、报告生成和报告查看。
- 在开发者工具中上传版本，再到微信公众平台提交审核并发布。

## 服务说明

小程序使用现有 Netlify 服务保存问卷和生成报告。小程序端通过微信登录生成访问令牌，不依赖网页 Cookie；网站后台仍用于查看线索、报告和生成网站邀请码。
